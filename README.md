# HDOS - Home DOS  
**UNDER CONSTRUCTION!**

**stubs.asm** - stubs for unimplemented functions

**101clock.asm** - real-time clock with blinking colon

**boot.asm** - OS bootloader (first disk sector)

**calc.asm** - calculator with basic operations

**comm.asm** - command dispatcher (handles //commands)

**err.asm** - critical error handler (kernel panic)

**fat32.asm** - FAT32 filesystem stub (not working)

**gui.asm** - graphical user interface (in development)

**hfs.asm** - HDOS File System (in development)

**int.asm** - interrupt handlers (timer)

**kern.asm** - main kernel, entry point after bootloader

**notepad.asm** - text editor

**sh.asm** - command shell

**snake.asm** - Snake game
